
# 플립 카운터 (Split-Flap Display) 스타일 청취자 카운터

기존 LED 스타일을 **80년대 공항/역 전광판에서 볼 수 있던 기계식 플립 카운터**로 교체합니다. 라디오의 따뜻한 우드톤/크림색 팔레트에 맞춰 디자인합니다.

---

## 디자인 컨셉

```
정상 상태:                    444 상태:
┌───┬───┬───┬───┬───┐        ┌───┬───┬───┐
│ 1 │ , │ 2 │ 4 │ 7 │        │ 4 │ 4 │ 4 │  ← 빨간 배경
└───┴───┴───┴───┴───┘        └───┴───┴───┘
    LISTENERS                    LISTENERS
```

### 색상 팔레트 (라디오 테마 적용)
- **플랩 배경**: `#2A231E` (radio-grille) - 어두운 갈색
- **플랩 텍스트**: `#F4E4BC` (radio-face) - 크림색
- **플랩 테두리**: `#C5A059` (radio-accent) - 황동색
- **444 상태**: `#8B0000` (다크 레드) 배경 + 글리치

## 수정 파일

### 1. `src/components/ListenerCounter.tsx` (전체 재작성)

플립 카운터 UI 구현:
- 각 숫자를 개별 "플랩" 카드로 표시
- CSS로 플랩의 입체감 표현 (상단 하이라이트, 하단 그림자)
- 가운데 수평선으로 플립 디스플레이 특유의 분할선 표현
- 숫자 변경 시 `rotateX` 애니메이션으로 플립 효과

### 2. `src/index.css` 수정

플립 카운터 전용 스타일 추가:
- `.flip-card`: 개별 숫자 카드 스타일
- `.flip-card-inner`: 플립 애니메이션
- `.flip-card-glitch`: 444 상태의 빨간색 + 떨림 효과
- 가운데 분할선 효과

### 3. `src/hooks/useListenerCounter.ts` (유지)

기존 로직 그대로 사용 (타이밍, 증가량 등 변경 없음)

## 플립 카운터 세부 구현

### FlipDigit 컴포넌트

각 숫자를 개별 플랩으로 렌더링:

```
구조:
┌─────────────────┐  ← 상단 하이라이트 (radio-accent/20)
│                 │
│       4         │  ← 숫자 (크림색)
│─────────────────│  ← 가운데 분할선
│                 │
│                 │
└─────────────────┘  ← 하단 그림자
```

### 애니메이션 효과

숫자 변경 시:
1. 현재 숫자가 위로 플립되며 사라짐 (`rotateX(-90deg)`)
2. 새 숫자가 아래에서 플립되며 나타남 (`rotateX(0deg)`)
3. 소리 없이 시각적 효과만 적용

### 444 상태 효과

- 플랩 배경색이 `#8B0000` (다크 레드)로 변경
- 전체 컨테이너에 미세한 떨림 효과
- 분할선이 빨간색으로 변경

## 기대 효과

80년대 공항이나 기차역에서 볼 수 있던 기계식 플립 디스플레이 느낌을 재현합니다. 숫자가 "딸깍" 하고 넘어가는 듯한 시각적 효과가 레트로 라디오 분위기와 잘 어울리며, 444로 떨어질 때 플랩이 빨갛게 변하면서 공포감을 줍니다.
